# Habit Tracker

A simple interactive habit tracking web app built with vanilla JavaScript to practice state management, DOM manipulation, and persistence.

## Live Demo
https://khzrs.github.io/habit-tracker/

## Features
- Create habits
- Mark habits complete for the day
- Automatic streak tracking
- Daily completion reset
- Delete habits
- Data persistence using localStorage

## Tech Used
- HTML
- CSS
- Vanilla JavaScript
- LocalStorage

## What I Practiced
- DOM manipulation
- Event delegation
- State-driven UI rendering
- Data persistence
- Git workflow and deployment with GitHub Pages
